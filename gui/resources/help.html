Adding

add <title> [on <start> to <end>] [by <end>] [#<tags>] [“<description>”]

Title

Name of the task to be added

Start

Optional start date and time

End

Optional end date and time (if only one date is to be inputted, an end date is expected)

Tags

Optional categories to add to

Description   Optional text that can include details of the text

Friendly reminder!

Make sure your start/end fields are after title, otherwise OraCle will not be able to read your command properly. The order of tags and description does not matter though!

Deleting

delete <ID>

ID         ID of the task to be deleted

Useful tips!

Think delete takes too long to type for a busy person like you? Try X instead! For example, type X 3 to delete task 3.

To make using OraCle a breeze, we offer shortcuts for other commands as well, as listed in

2.3 Reference List!

Undo-ing

undo

Undoes the last add or delete command

Shortcut        Ctrl+Z

Searching

show [<ID>] [<keywords>] [on <start> to <end>] [#<tags>] [status]

ID

Keywords

ID of the task you would like to view details of

Shows tasks that match the keyword within the <title> or <description> 

Start/End

Shows tasks that match the <start> or <end> date respectively

Tags                 Shows tasks that match the <tags> inputted

Status            Shows tasks that match the status needed (todo, done)

Shortcut        Ctrl+F

Changing task status

flag status <ID>

ID        ID of the task

Status        Status of the task to flag as. Valid statuses are: todo, done

Useful tips!

Alternatively, just type V <ID> to mark a task as done! For example, if you’ve completed your homework (with task ID 3), simply type V 3.

Editing

edit <ID> <new data 1> <new data 2> …..

ID

ID of the task to be edited

New data         Data to replace existing details, of type title, tags or description (same format as in add)

Set location

set <object> <name>

Object                 Object to be changed. Valid objects are: folder, avatar 

New details        File directory (for folder) or image file name (for avatar)
